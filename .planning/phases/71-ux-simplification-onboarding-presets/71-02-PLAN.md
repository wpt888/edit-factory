Add a preset selection grid at the top of the SubtitleEditor component, before the existing manual controls:

3. **Add preset grid section** at the top of the returned JSX (before the "Subtitle Style" heading):
   - A heading: "Style Presets" with a small description "Click a preset to apply"
   - A grid of clickable thumbnail cards (3 columns on desktop, 2 on mobile): `grid grid-cols-2 md:grid-cols-3 gap-2`
   - Each thumbnail card:
     - Background color from preset.previewStyle.backgroundColor
     - The preset.previewStyle.textSample rendered IN the preset's own style (font, color, outline simulated via text-shadow CSS)
     - The preset name below the preview area
     - A ring/border highlight when selected (ring-2 ring-primary)
     - Height: h-20 for a compact thumbnail feel
     - Rounded corners, cursor-pointer, hover:opacity-90 transition

4. **On preset click:**
   - Call `onSettingsChange({ ...settings, ...preset.settings })` to apply all preset settings at once
