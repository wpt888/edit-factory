**LD-04: Atomic install layout — exact paths**
- `INSTALL_ROOT = get_base_dir() / "ml"` — re-evaluate per-request (do not cache at module import; matches Phase 84 D-02 rationale).
- Partial download: `INSTALL_ROOT / ".partial" / <filename>`
- Staging unpack: `INSTALL_ROOT / ".staging" / <version>/`  (e.g. `.staging/0.1.0/`)
- Final marker: `INSTALL_ROOT / ".installed"` — contents = single line, the version string (e.g. `0.1.0\n`).

**LD-05: SSE event schema — exact event names + payloads**
Use `sse_starlette.sse.EventSourceResponse` driven by an `async def _event_stream()` generator. Events MUST yield in this order:
- `{"event": "progress", "data": json.dumps({"stage": "download", "downloaded": N, "total": M, "percent": P})}` — emit every ≥ 512 KB downloaded, AND at the start (0%) AND at the end (100%) of the download stage.
- `{"event": "progress", "data": json.dumps({"stage": "verify"})}` — emit once when SHA256 verification begins.
- `{"event": "progress", "data": json.dumps({"stage": "unpack"})}` — emit once when tar extraction begins.
- Terminal success: `{"event": "done", "data": json.dumps({"status": "installed", "version": ML_BUNDLE_VERSION})}` — emit ONCE after `.installed` is written.
- Terminal error: `{"event": "error", "data": json.dumps({"error": "<message>", "stage": "download" | "verify" | "unpack"})}` — emit on any caught Exception, then return from the generator. Do NOT raise.

**LD-06: HTTP Range resume — three branches**
Before downloading, stat `INSTALL_ROOT / ".partial" / <filename>`:
- If file does not exist: open new file with mode `"wb"`, send NO Range header.
- If file exists with size > 0: send `Range: bytes={existing_size}-`. Then on response:
  - Server returns `206 Partial Content`: open `"ab"` mode, append. (`Content-Length` is the REMAINING bytes; `total = existing_size + content_length`.)
  - Server returns `200 OK` (ignored Range): delete the partial, open `"wb"` mode, restart from byte 0.
  - Server returns `416 Range Not Satisfiable`: assume file is already complete — skip download, proceed directly to the verify stage against the existing `.partial/<filename>`.
  - Server returns any other status (e.g. 404, 500): emit `event: error` with stage=download and the status code in the message; return.

**LD-07: SHA256 verification**
- Compute SHA256 incrementally during download (`hashlib.sha256()` updated on every chunk write). On resume, when reading existing partial bytes from disk via the 416 path, hash them too.
- Fetch the `.sha256` sibling via `httpx.get(<base_url>/<filename>.sha256)`. Parse: `expected_hex = response.text.split()[0]` (first whitespace-delimited token).
- Compare `hasher.hexdigest()` to `expected_hex`. Case-insensitive.
- Mismatch: delete `.partial/<filename>`, emit `event: error` with stage=verify. Do NOT proceed to unpack.

**LD-08: Atomic unpack + promote**
- Use `tarfile.open(<partial>, "r:gz")` — extract into `INSTALL_ROOT / ".staging" / <version>/`. If `.staging/<version>` already exists, `shutil.rmtree` it first (idempotent retry).
- After extraction succeeds, atomically promote: any existing `INSTALL_ROOT` contents OTHER than `.partial`, `.staging`, and `.installed` are removed; then move every entry from `.staging/<version>/` into `INSTALL_ROOT` via `shutil.move`. Then `shutil.rmtree(.staging/<version>)`.
- Write `.installed` last: `(INSTALL_ROOT / ".installed").write_text(f"{ML_BUNDLE_VERSION}\n", encoding="utf-8")`.
- Delete `.partial/<filename>` only AFTER `.installed` is written successfully.

**LD-09: Streaming chunk size + memory cap**
- Chunk size: `CHUNK_SIZE = 1024 * 1024` (1 MB) — `async for chunk in response.aiter_bytes(CHUNK_SIZE)`.
- Do NOT load the response body into memory. Do NOT use `response.read()` or `response.content`. Stream chunks straight to disk.
- Incremental SHA256: `hasher.update(chunk)` before `f.write(chunk)`.
- Progress events emitted at most every 512 KB AND at most every 250 ms (use a `last_emit_bytes` + `last_emit_time` counter to throttle). This prevents spamming the SSE stream on fast networks.

**LD-10: Concurrent invocation guard**
- Module-level `_download_lock = asyncio.Lock()` and module-level `_download_in_progress: bool = False`.
- At route entry, atomically check `_download_in_progress`: if True, return `JSONResponse(status_code=409, content={"error": "download_in_progress"})` — no SSE.
- If False, set True before yielding the EventSourceResponse, and reset to False in a `finally` clause around the entire event-stream generator.
- The lock guards the check-and-set, not the long-running download itself.

**LD-11: HTTP client**
- Use `httpx.AsyncClient(follow_redirects=True, timeout=httpx.Timeout(connect=30.0, read=300.0, write=30.0, pool=None))` — GitHub Releases redirects to S3, so `follow_redirects=True` is required. 300 s read timeout accommodates slow chunks; per-chunk timeout is what matters, not total request time.

**LD-12: Test isolation**
- All tests in `tests/test_desktop_ml_routes.py` MUST mock `httpx.AsyncClient` so NO real HTTP request goes out. Use `unittest.mock.AsyncMock` or `httpx.MockTransport` (advisor recommendation: `httpx.MockTransport` is cleanest — pass it via `httpx.AsyncClient(transport=...)` in tests).
- Use `tmp_path` as `<base_dir>` by patching `app.api.desktop_ml_routes.get_base_dir` to return `tmp_path`.

    from fastapi.responses import JSONResponse
    from sse_starlette.sse import EventSourceResponse

    from app.config import get_base_dir

    logger = logging.getLogger(__name__)
    router = APIRouter(prefix="/desktop/ml", tags=["Desktop ML"])

    ML_BUNDLE_VERSION = "0.1.0"
    CHUNK_SIZE = 1024 * 1024  # 1 MB

    PROGRESS_THROTTLE_SECS = 0.25

    _download_lock = asyncio.Lock()
    _download_in_progress: bool = False


    def _resolve_bundle_filename() -> str:
        """Return the platform-specific bundle filename. Raises HTTPException(400) on unknown platforms."""
        version = ML_BUNDLE_VERSION
        if sys.platform == "win32":
            return f"ml-bundle-win64-{version}.tar.gz"
        if sys.platform == "darwin":
            if platform.machine() == "arm64":

            return f"ml-bundle-darwin-x64-{version}.tar.gz"
        if sys.platform.startswith("linux"):

        raise HTTPException(status_code=400, detail=f"Unsupported platform: {sys.platform}")


    def _resolve_base_url() -> str:
        """Return the bundle base URL with {version} substituted."""
        template = os.getenv(
            "ML_BUNDLE_BASE_URL",
            "https://github.com/wpt888/edit_factory/releases/download/ml-v{version}/",
        )
        return template.format(version=ML_BUNDLE_VERSION)


    @router.post("/download")
    async def download_ml_bundle():
        """POST /api/v1/desktop/ml/download — stream install progress as SSE."""
        global _download_in_progress
        async with _download_lock:
            if _download_in_progress:
                return JSONResponse(status_code=409, content={"error": "download_in_progress"})
            _download_in_progress = True
        # Resolve filename FIRST — unsupported platform must 400 before SSE starts.
        filename = _resolve_bundle_filename()
        base_url = _resolve_base_url()
        return EventSourceResponse(_event_stream(filename, base_url))

        global _download_in_progress

            partial_dir = install_root / ".partial"

    async def _download_with_progress(url: str, partial_path: Path):
        """Async generator. Yields dicts like {"event": "progress", "data": "..."}.
        After all chunks are written, yields a SINGLE sentinel dict:
        {"event": "__complete__", "downloaded": int, "total": int}.
        Caller filters out the sentinel and uses it to bridge to the verify stage.
        Implements 3-branch Range resume per LD-06.
        """
        existing_size = partial_path.stat().st_size if partial_path.exists() else 0
        headers = {}
        mode = "wb"
        if existing_size > 0:
            headers["Range"] = f"bytes={existing_size}-"

        timeout = httpx.Timeout(connect=30.0, read=300.0, write=30.0, pool=None)
        async with httpx.AsyncClient(follow_redirects=True, timeout=timeout) as client:
            async with client.stream("GET", url, headers=headers) as response:
                status = response.status_code
                if status == 416:
                    # File already complete — emit end-of-download progress with the existing size and return.
                    yield {"event": "__complete__", "downloaded": existing_size, "total": existing_size}
                    return
                if status == 206:
                    mode = "ab"
                    content_length = int(response.headers.get("Content-Length", "0") or "0")
                    expected_total = existing_size + content_length
                    downloaded = existing_size
                elif status == 200:
                    # Server ignored Range. Restart from byte 0.
                    if partial_path.exists():
                        partial_path.unlink()
                    mode = "wb"
                    content_length = int(response.headers.get("Content-Length", "0") or "0")
                    expected_total = content_length
                    downloaded = 0
                else:
                    raise RuntimeError(f"Unexpected status {status} from {url}")

                last_emit_bytes = downloaded
                last_emit_time = time.monotonic()
                yield {
                    "event": "progress",
                    "data": json.dumps({
                        "stage": "download",
                        "downloaded": downloaded,
                        "total": expected_total,
                        "percent": int((downloaded / expected_total * 100) if expected_total else 0),
                    }),
                }

                with open(partial_path, mode) as f:
                    async for chunk in response.aiter_bytes(CHUNK_SIZE):
                        f.write(chunk)
                        downloaded += len(chunk)
                        now = time.monotonic()
                        if (downloaded - last_emit_bytes >= PROGRESS_THROTTLE_BYTES) and (now - last_emit_time >= PROGRESS_THROTTLE_SECS):
                            yield {
                                "event": "progress",
                                "data": json.dumps({
                                    "stage": "download",
                                    "downloaded": downloaded,
                                    "total": expected_total,
                                    "percent": int((downloaded / expected_total * 100) if expected_total else 0),
                                }),
                            }
                            last_emit_bytes = downloaded
                            last_emit_time = now

                yield {"event": "__complete__", "downloaded": downloaded, "total": expected_total}


    async def _fetch_expected_sha256(url: str) -> str:
        timeout = httpx.Timeout(connect=30.0, read=60.0, write=30.0, pool=None)
        async with httpx.AsyncClient(follow_redirects=True, timeout=timeout) as client:
            response = await client.get(url)
            response.raise_for_status()
            text = response.text.strip()
            if not text:
                raise RuntimeError("empty sha256 file")
            return text.split()[0]


    def _hash_file_sha256(path: Path) -> str:
        hasher = hashlib.sha256()
        with open(path, "rb") as f:
            while True:
                chunk = f.read(CHUNK_SIZE)
                if not chunk:
                    break
                hasher.update(chunk)
        return hasher.hexdigest()


    def _unpack_and_promote(partial_path: Path, staging_dir: Path, install_root: Path) -> None:
        """Extract tarball to staging_dir, then atomically promote into install_root.
        Removes existing staging_dir contents first (idempotent retry).
        Removes any existing install_root contents EXCEPT .partial, .staging, .installed before promote.
        """
        if staging_dir.exists():
            shutil.rmtree(staging_dir)
        staging_dir.mkdir(parents=True, exist_ok=True)
        with tarfile.open(partial_path, "r:gz") as tar:
            tar.extractall(staging_dir)

        # Promote: remove existing install_root entries except managed dirs/marker
        for entry in install_root.iterdir():
            if entry.name in (".partial", ".staging", ".installed"):
                continue
            if entry.is_dir():
                shutil.rmtree(entry)
            else:
                entry.unlink()

        # Move staging contents into install_root
        for entry in staging_dir.iterdir():
            shutil.move(str(entry), str(install_root / entry.name))

        shutil.rmtree(staging_dir)
