## Goal

Add a "Crash Reporting" card to the Settings page with a Switch toggle that calls `POST /api/v1/desktop/crash-reporting` for immediate runtime effect. Only visible in desktop mode (`NEXT_PUBLIC_DESKTOP_MODE === "true"`).

## Context

- Settings page: `frontend/src/app/settings/page.tsx`
- `GET /desktop/settings` already returns `crash_reporting_enabled` (boolean) — see `desktop_routes.py` line 102
- Plan 51-01 adds `POST /desktop/crash-reporting` accepting `{ enabled: boolean }` and returning `{ crash_reporting_enabled: boolean }`
- The existing desktop version fetch (lines 244-250) already checks `NEXT_PUBLIC_DESKTOP_MODE`
- The Setup Wizard card (lines 1023-1037) is already gated on `NEXT_PUBLIC_DESKTOP_MODE`
- Shadcn Switch component: `@/components/ui/switch` (if not present, use a standard HTML checkbox styled as toggle)
- Use `apiPost` from `@/lib/api` for the POST call
- Follow existing page patterns: useState, useEffect, apiGetWithRetry

## Tasks

<task id="1">
<title>Add crash reporting state and load from GET /desktop/settings</title>
<details>
In `frontend/src/app/settings/page.tsx`:

1. Add state variable near the existing `appVersion` state (around line 111):
   ```typescript
   const [crashReporting, setCrashReporting] = useState(false)
   const [crashReportingLoading, setCrashReportingLoading] = useState(false)
   ```

2. Extend the existing desktop version `useEffect` (lines 244-250) to ALSO fetch crash reporting state. Modify the effect to:
   ```typescript
   useEffect(() => {
     if (process.env.NEXT_PUBLIC_DESKTOP_MODE !== 'true') return
     apiGetWithRetry('/desktop/version')
       .then((res) => res.json())
       .then((data: any) => setAppVersion(data.version))
       .catch(() => {})
     apiGetWithRetry('/desktop/settings')
       .then((res) => res.json())
       .then((data: any) => setCrashReporting(data.crash_reporting_enabled ?? false))
       .catch(() => {})
   }, [])
   ```

3. Add toggle handler:
   ```typescript
   const handleCrashReportingToggle = async (enabled: boolean) => {
     setCrashReportingLoading(true)
     setCrashReporting(enabled) // Optimistic update
     try {
       const res = await apiPost('/desktop/crash-reporting', { enabled })
       if (!res.ok) {
         setCrashReporting(!enabled) // Revert on error
         toast.error('Failed to update crash reporting setting')
       }
     } catch {
       setCrashReporting(!enabled) // Revert on error
       toast.error('Failed to update crash reporting setting')
     } finally {
       setCrashReportingLoading(false)
     }
   }
   ```
</details>
</task>
