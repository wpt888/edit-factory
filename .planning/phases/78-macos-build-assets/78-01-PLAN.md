<task type="auto">
  <name>Task 1: Create ICNS generator script and generate icon.icns</name>
  <files>electron/build/generate-icns.js, electron/build/icon.icns</files>
  <action>
Create `electron/build/generate-icns.js` that generates a valid macOS .icns file using pure Node.js (no native dependencies), following the same pattern as the existing `generate-icon.js`.

1. Copy the `generatePixels(size)` function and brand color constants from `generate-icon.js` (the rounded video frame with play triangle design).

2. Implement ICNS binary format writing:
   - ICNS files start with a header: magic bytes 'icns' (4 bytes) + total file size (4 bytes big-endian)
   - Followed by icon entries, each with: OSType (4 bytes) + entry size including header (4 bytes big-endian) + raw ARGB pixel data
   - Use these OSType codes for uncompressed ARGB data:
     - 'ic04' = 16x16 (not standard — skip, use 'is32' + 's8mk' pair or just skip 16px)
     - 'ic07' = 128x128
     - 'ic08' = 256x256
     - 'ic09' = 512x512
     - 'ic10' = 1024x1024 (retina 512@2x)
   - IMPORTANT: Modern ICNS files embed PNG data for sizes >= 128. Instead of raw ARGB, write PNG-encoded image data for each size. Use a minimal pure-JS PNG encoder (deflate via zlib which is built into Node.js).
   - For each size: render pixels via generatePixels(size), encode as PNG (RGBA, create IHDR + IDAT with zlib deflate + IEND), wrap in ICNS entry with appropriate OSType.

3. Generate these ICNS icon types (all as embedded PNG):
   - 'ic07' (128x128)
   - 'ic08' (256x256)
   - 'ic09' (512x512)
   - 'ic10' (1024x1024, used as 512@2x retina)
   - 'ic11' (32x32, used as 16@2x retina)
   - 'ic12' (64x64, used as 32@2x retina)
   - 'ic13' (256x256, used as 128@2x retina)
   - 'ic14' (512x512, used as 256@2x retina)

4. Write the assembled ICNS buffer to `electron/build/icon.icns`.

5. Use Node.js built-in `zlib.deflateSync` for PNG compression — no external dependencies needed.

6. Add a usage comment at top: `// Usage: node electron/build/generate-icns.js`

7. Run the script after creating it: `node electron/build/generate-icns.js`
