<task type="auto">
  <name>Task 2: Create build orchestration script</name>
  <files>scripts/build-installer.js</files>
  <action>
Create `scripts/build-installer.js` at the project root. This script orchestrates the full installer build process. It must run on Windows (native or WSL calling powershell).

The script sequences three steps:

**Step 1: Build Next.js standalone**
- Run `npm run build` in the `frontend/` directory (this triggers next build + postbuild.js which copies static assets)
- If build fails, exit with error

**Step 2: Download and extract portable Node.js 22 LTS**
- Check if `electron/resources/node/node.exe` already exists — skip if cached
- Download from `https://nodejs.org/dist/v22.22.0/node-v22.22.0-win-x64.zip`
- Use Node.js `https` module with redirect following (nodejs.org redirects)
- Save to `electron/resources/node-v22.22.0-win-x64.zip`
- Create `electron/resources/` directory if needed
- Extract using PowerShell `Expand-Archive` (Windows-only is acceptable — this builds Windows installers)
- Move contents of inner directory (`node-v22.22.0-win-x64/`) up to `electron/resources/node/`
- Clean up temp directory and zip file
- Verify `electron/resources/node/node.exe` exists after extraction

**Step 3: Run electron-builder**
- Run `npx electron-builder --win` in the `electron/` directory
- Use `stdio: 'inherit'` so build output is visible

**Important implementation details:**
- Use `const { execSync } = require('child_process')` for subprocess calls
- Use synchronous operations throughout (this is a build script, not a server)
- For the download step, since we need async https, wrap it in a self-executing async function or use `execSync` with `curl` or `powershell Invoke-WebRequest` instead of Node https (simpler for a build script)
- Actually, prefer `powershell Invoke-WebRequest` for download since we already use PowerShell for extraction — keeps it consistent and handles redirects automatically:
  ```javascript
  execSync(
    `powershell -command "Invoke-WebRequest -Uri '${NODE_ZIP_URL}' -OutFile '${NODE_ZIP}'"`,
    { stdio: 'inherit', timeout: 300000 }
  );
  ```
- Add a shebang `#!/usr/bin/env node` at the top
- Print clear step markers: `[build] Step 1: Building frontend...`, `[build] Step 2: Downloading portable Node.js...`, `[build] Step 3: Running electron-builder...`
- Print final message: `[build] Complete! Installer at electron/dist/`

**Error handling:**
- Wrap each step in try/catch
- On failure, print which step failed and exit with code 1
- Before Step 3, verify all required resources exist:
  - `electron/resources/node/node.exe`
  - `frontend/.next/standalone/server.js`
  - `venv/Scripts/python.exe`
  All paths relative to project root. If any missing, print clear error and exit.
  </action>
