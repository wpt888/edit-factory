<task id="1">
**Profile context caching with 60-second TTL**

In `app/api/auth.py`:

1. Add module-level cache structures at the top of the file (after imports):
   ```python
   import time as _time
   import threading as _threading

   # Profile context cache: (user_id, profile_id_or_default) -> (ProfileContext, timestamp)
   _profile_cache: Dict[tuple, tuple] = {}
   _profile_cache_lock = _threading.Lock()
   _PROFILE_CACHE_TTL = 60  # seconds
   ```
   Add `Dict` to the typing import.

2. Add helper functions:
   ```python
   def _cache_get_profile(user_id: str, profile_key: str) -> Optional[ProfileContext]:
       """Return cached ProfileContext if TTL not expired, else None."""
       with _profile_cache_lock:
           entry = _profile_cache.get((user_id, profile_key))
           if entry is None:
               return None
           ctx, ts = entry
           if _time.monotonic() - ts > _PROFILE_CACHE_TTL:
               del _profile_cache[(user_id, profile_key)]
               return None
           return ctx

   def _cache_set_profile(user_id: str, profile_key: str, ctx: ProfileContext) -> None:
       """Store ProfileContext in cache with current timestamp."""
       with _profile_cache_lock:
           _profile_cache[(user_id, profile_key)] = (ctx, _time.monotonic())
   ```

3. Modify `get_profile_context()`:
   - **Dev mode path (auth_disabled/desktop_mode):** After determining the profile_id but BEFORE returning, check cache with `_cache_get_profile(current_user.id, x_profile_id or "default")`. If hit, return cached value. If miss, proceed with Supabase query and store result via `_cache_set_profile`.
   - **Production path (no x_profile_id):** Before the Supabase query for default profile, check `_cache_get_profile(current_user.id, "default")`. If hit, return cached. If miss, query Supabase, store with `_cache_set_profile(current_user.id, "default", ctx)`, then return.
   - **Production path (with x_profile_id):** Check `_cache_get_profile(current_user.id, x_profile_id)`. If hit, return. If miss, do the ownership validation query, store, return.
   - **NEVER cache error states.** Only cache after successful ProfileContext creation.
   - **Bounded cache size:** If `len(_profile_cache) > 1000`, clear all entries (simple eviction to prevent unbounded growth in long-running processes).

**Verification:** Log or debug-print cache hits. Make 10 rapid requests to any authenticated endpoint — only the first should trigger a Supabase profile query.

3. Change `_evict_if_needed()` from oldest-mtime to LRU:
   - Sort by `st_atime` (access time) instead of `st_mtime`:
     ```python
     cache_files = sorted(cache_dir.glob("*.mp3"), key=lambda f: f.stat().st_atime)
     ```
   - This means least-recently-accessed files get evicted first

4. Add `cache_stats()` function:
   ```python
   def cache_stats() -> dict:
       """Return TTS cache statistics."""
       root = _get_cache_root()
       current_size = 0
       try:
           for provider_dir in root.iterdir():
               if provider_dir.is_dir():
                   current_size += len(list(provider_dir.glob("*.mp3")))
       except Exception:
           pass

       with _stats_lock:
           return {
               "hit_count": _hit_count,
               "miss_count": _miss_count,
               "current_size": current_size,
               "max_size": MAX_CACHE_ENTRIES,
           }
   ```
