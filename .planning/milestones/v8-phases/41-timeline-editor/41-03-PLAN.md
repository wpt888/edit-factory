Update the component props to use `ExtendedMatch[]` or handle the field as optional on MatchPreview.

**2. Duration control UI per row:**

Add a compact duration display and adjustment on each row:
- Show current duration: `{formatTime(match.duration_override || (match.srt_end - match.srt_start))}s`
- Add +/- buttons (0.5s increments) beside the duration display
- Minimum duration: 0.5s, maximum duration: 10s
- When adjusted, add/update `duration_override` on the match and call `onMatchesChange`

Use a compact inline layout:

**3. adjustDuration handler:**
```typescript
const adjustDuration = (index: number, delta: number) => {
  const match = matches[index];
  const currentDuration = match.duration_override || (match.srt_end - match.srt_start);
  const newDuration = Math.max(0.5, Math.min(10, currentDuration + delta));
  const updated = [...matches];
  updated[index] = { ...updated[index], duration_override: newDuration };
  onMatchesChange(updated);
};
```

**4. Visual indicator when duration is overridden:**
Show the duration in a different color (blue) when it differs from the natural SRT duration, to indicate manual adjustment.
